### Downloading the Laboratories ###

To download and use the laboratories, you need to install a tool called Git.
You can download it from the following link:

[Git Downloads](https://git-scm.com/install/windows)

Install Git and open a terminal in the folder where you want to work where you want to work and run:

```bash
git clone https://github.com/JeroHoyos/ML_Seminar
cd Lab01
```
